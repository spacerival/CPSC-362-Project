CPSC Group Project

Project Name: Budget Tracker

=== What is it? ===
A website application that helps users keep track of their finances and curate budget plans according to their needs. This project showcases the use of HTML, CSS, JavaScript, and Google Firebase for backend and database management. 

=== Website Features ===
- Create a profile, which allows you to save your previously logged transactions and budget plans.
- See an overview of you balance, income, expenses, and categorized spending
- Be able to add transactions (income/expenses), either individually or importing a CSV file
- Have all your transactions logged in one place, sorted by...
    - Type (All/Income Only/Expenses Only)
    - Category (Rent, Healthcare, Groceries, Utilities, etc.)
    - Date (Newest to Oldest/Oldest to Newest) or Amount (Low to High/High to Low)
- Choose from 3 budget plans according to your financial needs, and change plans if needed
    - 50/30/20
    - Zero-Based
    - Pay-Yourself-First
- Go to your profile page to change your name, email, or password
- Logout from your password page

=== How to Set Up Project === 
1. Go to the GitHub repository "CPSC-362-Project"
2. Click on the green button that says "<> Code"
3. Click "Download Zip"
4. Extract the folder from the downloaded zip file
5. Go to VScode (or your preferred IDE) and open the folder you just extracted. Make sure all the files are
included in the same folder.
6. Make sure you have a means of opening HTML files into your browser. On VScode, install the extension "Live Server" by Ritwick Dey or "Live Server(Five Server)" by Yannick either extension will work.
7. Open the "main.html" file in browser. If you have the VScode extensions previously mentioned, do so by clicking "Go Live" in the bottom right hand corner of VScode.
8. Now you should be able to explore the Budget Tracker website!
