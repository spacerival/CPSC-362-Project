CSV File Format Requirements

  • Your CSV must include these columns: type, amount, category, description, date
  • type: Must be either "income" or "expense"
  • amount: Numeric value (e.g., 50.00)
  • category: Must be one of the following:
    • Essentials/Needs: rent, utilities, healthcare, education, transportation, insurance, groceries, loans
    • Wants: shopping, entertainment, diningout, leisuretravel
    • Savings: savings, investment, retirement, emergency
    • Income: salary, freelance, business, gift
    • Other: other
  • description: Text description of the transaction
  • date: Date in format YYYY-MM-DD (e.g., 2025-01-15)
  • Optional column: notes for additional information
